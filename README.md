🤖 AI ChatBot

📌 Project Overview

The AI ChatBot is a Java console-based chatbot that provides interactive conversations using predefined responses and user inputs. The chatbot can remember the user's name during the session, answer common questions, provide date and time information, tell jokes, and perform simple calculations.

This project demonstrates the creation of an intelligent rule-based chatbot using Java programming concepts.

✨ Features

- Personalized user interaction using the user's name
- Greeting and conversation handling
- Random responses for more natural interaction
- Display current date and time
- Provide Java-related information
- Tell programming jokes
- Perform simple addition calculations
- Help command for available features
- Continuous chat until the user exits

🛠 Technologies Used

- Java
- Object-Oriented Programming Concepts
- Scanner Class
- Random Class
- Java Date & Time API
- Conditional Statements and Loops

📂 Project Structure

- AIChatBot Class – Contains chatbot logic and conversation handling

🚀 How to Run

1. Install Java JDK on your system.
2. Save the file as "AIChatBot.java".
3. Compile the program:
   "javac AIChatBot.java"
4. Run the application:
   "java AIChatBot"
